Ejercicio 1

Se tiene el vector $\vec{U} = (6, 3, x)$, calcule el valor de $x$ para que la magnitud del vector sea igual a 9.

📝 Desarrollo:

Paso 1: Aplicar la fórmula de magnitud
$$\|\vec{U}\| = \sqrt{6^2 + 3^2 + x^2} = 9$$ $$\sqrt{36 + 9 + x^2} = 9$$ $$\sqrt{45 + x^2} = 9$$
Paso 2: Elevar al cuadrado
$$45 + x^2 = 81$$ $$x^2 = 36$$ $$x = \pm 6$$
✅ Respuesta: $x = 6$ o $x = -6$

Ejercicio 2

Se tiene el vector $\vec{U} = (x, 9, 8)$, calcule el valor de $x$ para que la magnitud del vector sea igual a 17.

📝 Desarrollo:

Aplicar la fórmula
$$\sqrt{x^2 + 9^2 + 8^2} = 17$$ $$\sqrt{x^2 + 81 + 64} = 17$$ $$\sqrt{x^2 + 145} = 17$$ $$x^2 + 145 = 289$$ $$x^2 = 144$$ $$x = \pm 12$$
✅ Respuesta: $x = 12$ o $x = -12$

Ejercicio 3

Los vectores $\vec{U} = (-5, 7, 4)$ y $\vec{V} = (-3, 2, 6)$ forman un paralelogramo, calcular el área de dicha figura.

📝 Desarrollo:

Calcular el producto cruz
$$\vec{U} \times \vec{V} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ -5 & 7 & 4 \\ -3 & 2 & 6 \end{vmatrix}$$ $$\vec{i}: (7)(6) - (4)(2) = 42 - 8 = 34$$ $$\vec{j}: -[(-5)(6) - (4)(-3)] = -[-30 + 12] = 18$$ $$\vec{k}: (-5)(2) - (7)(-3) = -10 + 21 = 11$$ $$\vec{U} \times \vec{V} = (34, 18, 11)$$
Calcular la magnitud
$$\text{Área} = \|\vec{U} \times \vec{V}\| = \sqrt{34^2 + 18^2 + 11^2}$$ $$= \sqrt{1156 + 324 + 121} = \sqrt{1601}$$
✅ Área del paralelogramo: $\sqrt{1601}$ unidades² $\approx 40.01$ unidades²

Ejercicio 4

Calcular el volumen de un paralelepípedo determinado por los vectores: $\vec{U} = (5, -1, 8)$, $\vec{V} = (6, 7, -9)$ y $\vec{W} = (5, -3, 2)$ en $\mathbb{R}^{3}$.

📝 Desarrollo:

Calcular $\vec{V} \times \vec{W}$
$$\vec{V} \times \vec{W} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 6 & 7 & -9 \\ 5 & -3 & 2 \end{vmatrix}$$ $$= \vec{i}(14 - 27) - \vec{j}(12 + 45) + \vec{k}(-18 - 35)$$ $$= (-13, -57, -53)$$
Calcular el producto mixto
$$\text{Volumen} = |\vec{U} \cdot (\vec{V} \times \vec{W})|$$ $$= |(5)(-13) + (-1)(-57) + (8)(-53)|$$ $$= |-65 + 57 - 424|$$ $$= |-432| = 432$$
✅ Volumen: $432$ unidades³

Ejercicio 5

Calcule la distancia entre los puntos $P(3, 1, -4)$ y $Q(0, 1, 2)$.

📝 Desarrollo:

Aplicar la fórmula de distancia
$$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$$ $$d = \sqrt{(0-3)^2 + (1-1)^2 + (2-(-4))^2}$$ $$d = \sqrt{9 + 0 + 36} = \sqrt{45} = 3\sqrt{5}$$
✅ Distancia: $3\sqrt{5} \approx 6.71$ unidades

Ejercicio 6

Con base en las coordenadas del punto anterior, encuentra las coordenadas del punto que se encuentra a $\frac{1}{4}$ de la distancia entre $P$ y $Q$ a partir de $P$.

📝 Desarrollo:

Datos: $P(3, 1, -4)$ y $Q(0, 1, 2)$

🔵 Método 1: Usando parámetro t

Paso 1: Fórmula del punto en el segmento
$$R = P + t(Q - P) \text{ donde } t = \frac{1}{4}$$
Paso 2: Calcular $Q - P$
$$Q - P = (0-3, 1-1, 2-(-4)) = (-3, 0, 6)$$
Paso 3: Calcular el punto R
$$R = (3, 1, -4) + \frac{1}{4}(-3, 0, 6)$$ $$R = (3, 1, -4) + (-0.75, 0, 1.5)$$ $$R = (2.25, 1, -2.5)$$

🔵 Método 2: Usando la razón (notación r:s)

Interpretación de la razón:
Si el punto R está a $\frac{1}{4}$ de la distancia de P a Q:
• Distancia de P a R: $\frac{1}{4}$ de PQ
• Distancia de R a Q: $\frac{3}{4}$ de PQ
• Razón r:s = 1:3
Fórmula usando la razón r:s:
$$R = \frac{s \cdot P + r \cdot Q}{r + s} = \frac{3P + 1Q}{1 + 3}$$
Sustituir valores:
$$R = \frac{3(3, 1, -4) + 1(0, 1, 2)}{4}$$ $$R = \frac{(9, 3, -12) + (0, 1, 2)}{4}$$ $$R = \frac{(9, 4, -10)}{4}$$ $$R = \left(\frac{9}{4}, \frac{4}{4}, \frac{-10}{4}\right)$$ $$R = \left(\frac{9}{4}, 1, -\frac{5}{2}\right)$$

🟡 Método 3: Usando la razón k = r/s (constante)

Definir la constante k:
Si la razón es r:s = 1:3, entonces:
$$k = \frac{r}{s} = \frac{1}{3}$$
📌 Fórmula para DIVISIÓN INTERNA (punto entre P y Q):
$$R = \frac{P + kQ}{1 + k}$$
Aplicando coordenada por coordenada:
$$x = \frac{x_1 + kx_2}{1 + k}, \quad y = \frac{y_1 + ky_2}{1 + k}, \quad z = \frac{z_1 + kz_2}{1 + k}$$
Sustituir k = 1/3:
$$x = \frac{3 + \frac{1}{3}(0)}{1 + \frac{1}{3}} = \frac{3}{\frac{4}{3}} = \frac{9}{4}$$ $$y = \frac{1 + \frac{1}{3}(1)}{1 + \frac{1}{3}} = \frac{1 + \frac{1}{3}}{\frac{4}{3}} = \frac{\frac{4}{3}}{\frac{4}{3}} = 1$$ $$z = \frac{-4 + \frac{1}{3}(2)}{1 + \frac{1}{3}} = \frac{-4 + \frac{2}{3}}{\frac{4}{3}} = \frac{-\frac{10}{3}}{\frac{4}{3}} = -\frac{5}{2}$$
⚠️ Fórmula para DIVISIÓN EXTERNA (punto fuera del segmento):
Si usas la fórmula con RESTA:
$$R = \frac{P - kQ}{1 - k} \quad \text{(solo para división externa)}$$
Esta NO es la fórmula para este problema, ya que R está ENTRE P y Q.
💡 Resumen de fórmulas con razón k:
• División INTERNA (punto entre P y Q): $R = \frac{P + kQ}{1 + k}$ ✅
• División EXTERNA (punto fuera del segmento): $R = \frac{P - kQ}{1 - k}$ ❌ (no aplica aquí)

Los tres métodos dan el mismo resultado para división interna.
✅ Respuesta: $R = \left(\frac{9}{4}, 1, -\frac{5}{2}\right)$ o $R = (2.25, 1, -2.5)$

Ejercicio 7

Dados los siguientes vectores (ver imagen):
  • Vector $\vec{a}$: apunta hacia abajo, de tamaño 2
  • Vector $\vec{b}$: apunta hacia la derecha, de tamaño 3
  • Vector $\vec{c}$: apunta hacia abajo-derecha (diagonal), de tamaño $\sqrt{18}$
a) $3\vec{a} - 2\vec{b}$
b) $\vec{b} \cdot \vec{c} + 2\vec{b}$

📝 Desarrollo:

Nota: Según la descripción, podemos establecer:
• $\vec{a} = (0, -2)$ (hacia abajo)
• $\vec{b} = (3, 0)$ (hacia derecha)
• $\vec{c} = (3, -3)$ (diagonal, magnitud $\sqrt{18} = 3\sqrt{2}$)
Parte a) $3\vec{a} - 2\vec{b}$
$$3\vec{a} = 3(0, -2) = (0, -6)$$ $$2\vec{b} = 2(3, 0) = (6, 0)$$ $$3\vec{a} - 2\vec{b} = (0, -6) - (6, 0) = (-6, -6)$$
Resultado: Vector que apunta hacia abajo-izquierda con magnitud $6\sqrt{2}$
Parte b) $\vec{b} \cdot \vec{c} + 2\vec{b}$
$$\vec{b} \cdot \vec{c} = (3, 0) \cdot (3, -3) = 9 + 0 = 9$$ $$2\vec{b} = 2(3, 0) = (6, 0)$$
Nota: $\vec{b} \cdot \vec{c}$ es un escalar (9), por lo que la expresión $9 + 2\vec{b}$ no está bien definida (no se puede sumar un escalar con un vector).

Interpretación alternativa: Si fuera $(\vec{b} \cdot \vec{c})\vec{b} + 2\vec{b}$:
$$9\vec{b} + 2\vec{b} = 11\vec{b} = 11(3, 0) = (33, 0)$$
✅ Respuestas:
a) $3\vec{a} - 2\vec{b} = (-6, -6)$
b) $\vec{b} \cdot \vec{c} = 9$ (escalar). Si el ejercicio requiere dibujo, verificar enunciado original.

Ejercicio 8

Encuentra la ecuación general del plano que contiene las siguientes rectas: $$l_{1}: \frac{x+2}{3} = \frac{3-y}{4} = \frac{z}{5}$$ $$l_{2}: \frac{x+5}{3} = \frac{y+2}{-4} = \frac{z-3}{2}$$

📝 Desarrollo:

Paso 1: Identificar puntos y vectores directores
$$l_1: P_1 = (-2, 3, 0), \quad \vec{v}_1 = (3, -4, 5)$$ $$l_2: P_2 = (-5, -2, 3), \quad \vec{v}_2 = (3, -4, 2)$$
Paso 2: Vector entre los dos puntos
$$\vec{P_1P_2} = P_2 - P_1 = (-5-(-2), -2-3, 3-0) = (-3, -5, 3)$$
Paso 3: Verificar que las rectas son coplanares Las rectas son coplanares si $\vec{P_1P_2} \cdot (\vec{v}_1 \times \vec{v}_2) = 0$
$$\vec{v}_1 \times \vec{v}_2 = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 3 & -4 & 5 \\ 3 & -4 & 2 \end{vmatrix}$$ $$= \vec{i}(-8+20) - \vec{j}(6-15) + \vec{k}(-12+12)$$ $$= (12, 9, 0)$$
$$\vec{P_1P_2} \cdot (\vec{v}_1 \times \vec{v}_2) = (-3, -5, 3) \cdot (12, 9, 0)$$ $$= -36 - 45 + 0 = -81 \neq 0$$
Las rectas NO son coplanares (son alabeadas), por lo tanto NO existe un plano que las contenga a ambas.
⚠️ Respuesta: Las rectas son alabeadas. No existe un plano que contenga ambas rectas.

Ejercicio 9

Encuentra la ecuación general del plano que contiene las siguientes rectas: $$l_{1}: \frac{4-x}{2} = y + 1 = \frac{z-1}{-2}$$ $$l_{2}: x - 1 = \frac{y+2}{2} = \frac{2-z}{3}$$

📝 Desarrollo:

Paso 1: Reescribir en forma estándar
$$l_1: \frac{x-4}{-2} = \frac{y+1}{1} = \frac{z-1}{-2}$$ $$P_1 = (4, -1, 1), \quad \vec{v}_1 = (-2, 1, -2)$$
$$l_2: \frac{x-1}{1} = \frac{y+2}{2} = \frac{z-2}{-3}$$ $$P_2 = (1, -2, 2), \quad \vec{v}_2 = (1, 2, -3)$$
Paso 2: Vector entre puntos
$$\vec{P_1P_2} = (1-4, -2-(-1), 2-1) = (-3, -1, 1)$$
Paso 3: Verificar coplanaridad
$$\vec{v}_1 \times \vec{v}_2 = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ -2 & 1 & -2 \\ 1 & 2 & -3 \end{vmatrix}$$ $$= \vec{i}(-3+4) - \vec{j}(6+2) + \vec{k}(-4-1)$$ $$= (1, -8, -5)$$
$$\vec{P_1P_2} \cdot (\vec{v}_1 \times \vec{v}_2) = (-3, -1, 1) \cdot (1, -8, -5)$$ $$= -3 + 8 - 5 = 0 \quad ✓$$
Las rectas SÍ son coplanares
Paso 4: Vector normal al plano El vector normal es $\vec{n} = \vec{v}_1 \times \vec{v}_2 = (1, -8, -5)$
Paso 5: Ecuación del plano
$$1(x-4) - 8(y+1) - 5(z-1) = 0$$ $$x - 4 - 8y - 8 - 5z + 5 = 0$$ $$x - 8y - 5z - 7 = 0$$
✅ Ecuación del plano: $x - 8y - 5z = 7$

Ejercicio 10

Encuentra la ecuación general del plano que contiene las siguientes rectas paralelas: $$l_{1}: \frac{x-5}{5} = \frac{y+3}{-4} = \frac{z-4}{3}$$ $$l_{2}: \frac{x-3}{15} = \frac{y-2}{-12} = \frac{z-1}{9}$$

Ejercicio 12

Calcular el ángulo que se forma entre los planos $\pi_{1}: 3x - 5y - 4z = 2$ y $\pi_{2}: -3x + 2y - 3z = -4$

📝 Desarrollo:

Paso 1: Vectores normales
$$\vec{n}_1 = (3, -5, -4)$$ $$\vec{n}_2 = (-3, 2, -3)$$
Paso 2: Fórmula del ángulo
$$\cos \theta = \frac{|\vec{n}_1 \cdot \vec{n}_2|}{\|\vec{n}_1\| \|\vec{n}_2\|}$$
Paso 3: Producto escalar
$$\vec{n}_1 \cdot \vec{n}_2 = (3)(-3) + (-5)(2) + (-4)(-3)$$ $$= -9 - 10 + 12 = -7$$ $$|\vec{n}_1 \cdot \vec{n}_2| = 7$$
Paso 4: Magnitudes
$$\|\vec{n}_1\| = \sqrt{3^2 + (-5)^2 + (-4)^2} = \sqrt{9 + 25 + 16} = \sqrt{50} = 5\sqrt{2}$$ $$\|\vec{n}_2\| = \sqrt{(-3)^2 + 2^2 + (-3)^2} = \sqrt{9 + 4 + 9} = \sqrt{22}$$
Paso 5: Calcular ángulo
$$\cos \theta = \frac{7}{5\sqrt{2} \cdot \sqrt{22}} = \frac{7}{5\sqrt{44}} = \frac{7}{10\sqrt{11}}$$ $$\theta = \arccos\left(\frac{7}{10\sqrt{11}}\right)$$ $$\theta \approx \arccos(0.2108) \approx 77.83°$$
✅ Ángulo entre los planos: $\theta = \arccos\left(\frac{7\sqrt{11}}{110}\right) \approx 77.83°$ o $\approx 1.358$ radianes

Ejercicio 13

Dados el punto $P(2, -5, 4)$ y la recta $l: \frac{x+2}{5} = \frac{10y-5}{15} = \frac{2-z}{4}$, verifique que el punto no está contenido en la recta y calcule la distancia entre el punto y la recta.

📝 Desarrollo:

Paso 1: Simplificar la recta
$$\frac{x+2}{5} = \frac{10(y-0.5)}{15} = \frac{2-z}{4}$$ $$\frac{x+2}{5} = \frac{2(y-0.5)}{3} = \frac{z-2}{-4}$$
Simplificando:
$$\frac{x+2}{5} = \frac{y-0.5}{1.5} = \frac{z-2}{-4}$$
Punto en la recta: $Q = (-2, 0.5, 2)$
Vector director: $\vec{v} = (5, 1.5, -4)$ o simplificado $\vec{v} = (10, 3, -8)$
Paso 2: Verificar si P está en la recta Si $P$ está en la recta: $\frac{2+2}{5} = \frac{10(-5)-5}{15} = \frac{2-4}{4}$
$$\frac{4}{5} \neq \frac{-55}{15} \neq \frac{-2}{4}$$
El punto NO está en la recta ✓
Paso 3: Calcular distancia Fórmula: $d = \frac{\|\vec{QP} \times \vec{v}\|}{\|\vec{v}\|}$
$$\vec{QP} = (2-(-2), -5-0.5, 4-2) = (4, -5.5, 2)$$
$$\vec{QP} \times \vec{v} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 4 & -5.5 & 2 \\ 10 & 3 & -8 \end{vmatrix}$$ $$= \vec{i}(44-6) - \vec{j}(-32-20) + \vec{k}(12+55)$$ $$= (38, 52, 67)$$
$$\|\vec{QP} \times \vec{v}\| = \sqrt{38^2 + 52^2 + 67^2} = \sqrt{1444 + 2704 + 4489} = \sqrt{8637}$$ $$\|\vec{v}\| = \sqrt{10^2 + 3^2 + (-8)^2} = \sqrt{100 + 9 + 64} = \sqrt{173}$$ $$d = \frac{\sqrt{8637}}{\sqrt{173}} = \sqrt{\frac{8637}{173}} = \sqrt{49.925...} \approx 7.07$$
✅ Distancia: $d = \frac{\sqrt{8637}}{\sqrt{173}} \approx 7.07$ unidades

Ejercicio 15

Dados el punto $P(2, -5, 4)$ y el plano $\pi: 2x - 8y + 6z = -4$, verifique que el punto no pertenece al plano y calcule la distancia entre el punto y el plano.

📝 Desarrollo:

Paso 1: Verificar si P está en el plano Sustituir $P(2, -5, 4)$ en la ecuación:
$$2(2) - 8(-5) + 6(4) = 4 + 40 + 24 = 68 \neq -4$$
El punto NO está en el plano ✓
Paso 2: Fórmula de distancia punto-plano
$$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$$
Donde el plano es $2x - 8y + 6z + 4 = 0$
Paso 3: Calcular distancia
$$d = \frac{|2(2) - 8(-5) + 6(4) + 4|}{\sqrt{2^2 + (-8)^2 + 6^2}}$$ $$= \frac{|4 + 40 + 24 + 4|}{\sqrt{4 + 64 + 36}}$$ $$= \frac{72}{\sqrt{104}} = \frac{72}{2\sqrt{26}} = \frac{36}{\sqrt{26}}$$ $$= \frac{36\sqrt{26}}{26} = \frac{18\sqrt{26}}{13} \approx 7.06$$
✅ Distancia: $d = \frac{18\sqrt{26}}{13} \approx 7.06$ unidades

Ejercicio 17

Dadas las rectas $l_{1}: \frac{x+8}{4} = \frac{y+1}{6} = \frac{z+5}{-1}$ y $l_{2}: \frac{x-1}{8} = \frac{y-3}{12} = \frac{z+2}{-2}$, verifique que las rectas son paralelas y calcule la distancia entre ellas.

📝 Desarrollo:

Paso 1: Identificar datos
$$l_1: P_1 = (-8, -1, -5), \quad \vec{v}_1 = (4, 6, -1)$$ $$l_2: P_2 = (1, 3, -2), \quad \vec{v}_2 = (8, 12, -2)$$

Las rectas son paralelas ✓
Paso 3: Distancia entre rectas paralelas Fórmula: $d = \frac{\|\vec{P_1P_2} \times \vec{v}_1\|}{\|\vec{v}_1\|}$
$$\vec{P_1P_2} = (1-(-8), 3-(-1), -2-(-5)) = (9, 4, 3)$$
$$\vec{P_1P_2} \times \vec{v}_1 = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 9 & 4 & 3 \\ 4 & 6 & -1 \end{vmatrix}$$ $$= \vec{i}(-4-18) - \vec{j}(-9-12) + \vec{k}(54-16)$$ $$= (-22, 21, 38)$$
$$\|\vec{P_1P_2} \times \vec{v}_1\| = \sqrt{484 + 441 + 1444} = \sqrt{2369}$$ $$\|\vec{v}_1\| = \sqrt{16 + 36 + 1} = \sqrt{53}$$ $$d = \frac{\sqrt{2369}}{\sqrt{53}} = \sqrt{\frac{2369}{53}} = \sqrt{44.698...} \approx 6.69$$
✅ Distancia entre las rectas paralelas: $d = \frac{\sqrt{2369}}{\sqrt{53}} \approx 6.69$ unidades

Ejercicio 18

Dadas las rectas $l_{1}: \frac{x-4}{-3} = \frac{y-3}{-2} = \frac{z-10}{-5}$ y $l_{2}: \frac{x-7}{6} = \frac{y+8}{4} = \frac{z-6}{10}$, verifique que las rectas son paralelas y calcule la distancia entre ellas.

📝 Desarrollo:

Paso 1: Identificar datos
$$l_1: P_1 = (4, 3, 10), \quad \vec{v}_1 = (-3, -2, -5)$$ $$l_2: P_2 = (7, -8, 6), \quad \vec{v}_2 = (6, 4, 10)$$
Paso 2: Verificar paralelismo
$$\vec{v}_2 = (6, 4, 10) = -2(-3, -2, -5) = -2\vec{v}_1$$
Las rectas son paralelas ✓
Paso 3: Calcular distancia
$$\vec{P_1P_2} = (7-4, -8-3, 6-10) = (3, -11, -4)$$
$$\vec{P_1P_2} \times \vec{v}_1 = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 3 & -11 & -4 \\ -3 & -2 & -5 \end{vmatrix}$$ $$= \vec{i}(55-8) - \vec{j}(-15-12) + \vec{k}(-6-33)$$ $$= (47, 27, -39)$$
$$\|\vec{P_1P_2} \times \vec{v}_1\| = \sqrt{2209 + 729 + 1521} = \sqrt{4459}$$ $$\|\vec{v}_1\| = \sqrt{9 + 4 + 25} = \sqrt{38}$$ $$d = \frac{\sqrt{4459}}{\sqrt{38}} = \sqrt{\frac{4459}{38}} = \sqrt{117.34...} \approx 10.83$$

Ejercicio 19

Dados el plano $\pi: 2x - 4y + 2z = -4$ y la recta $l: \frac{x+8}{4} = \frac{y+1}{1} = \frac{z+5}{-2}$, verificar si la recta es paralela al plano y calcule la distancia entre la recta y el plano.

📝 Desarrollo:

Paso 1: Identificar datos
$$\text{Plano: } \vec{n} = (2, -4, 2)$$ $$\text{Recta: } P = (-8, -1, -5), \quad \vec{v} = (4, 1, -2)$$
Paso 2: Verificar paralelismo La recta es paralela al plano si $\vec{n} \cdot \vec{v} = 0$
$$\vec{n} \cdot \vec{v} = (2)(4) + (-4)(1) + (2)(-2) = 8 - 4 - 4 = 0$$
La recta es paralela al plano ✓
Paso 3: Verificar que no está contenida Sustituir $P(-8, -1, -5)$ en el plano:
$$2(-8) - 4(-1) + 2(-5) = -16 + 4 - 10 = -22 \neq -4$$
La recta NO está en el plano
Paso 4: Distancia
$$d = \frac{|2(-8) - 4(-1) + 2(-5) - (-4)|}{\sqrt{2^2 + (-4)^2 + 2^2}}$$ $$= \frac{|-16 + 4 - 10 + 4|}{\sqrt{4 + 16 + 4}} = \frac{|-18|}{\sqrt{24}}$$ $$= \frac{18}{2\sqrt{6}} = \frac{9}{\sqrt{6}} = \frac{9\sqrt{6}}{6} = \frac{3\sqrt{6}}{2} \approx 3.67$$
✅ Distancia recta-plano: $d = \frac{3\sqrt{6}}{2} \approx 3.67$ unidades

Ejercicio 20

Dados el plano $\pi: 3x + 4y - 5z = 6$ y la recta $l: \frac{x+8}{-3} = \frac{y+1}{-4} = \frac{z+5}{-5}$, verificar si la recta es paralela al plano y calcule la distancia entre la recta y el plano.

📝 Desarrollo:

Paso 1: Identificar datos
$$\text{Plano: } \vec{n} = (3, 4, -5)$$ $$\text{Recta: } P = (-8, -1, -5), \quad \vec{v} = (-3, -4, -5)$$
Paso 2: Verificar paralelismo
$$\vec{n} \cdot \vec{v} = (3)(-3) + (4)(-4) + (-5)(-5)$$ $$= -9 - 16 + 25 = 0$$
La recta es paralela al plano ✓
Paso 3: Verificar que no está contenida
$$3(-8) + 4(-1) - 5(-5) = -24 - 4 + 25 = -3 \neq 6$$
La recta NO está en el plano
Paso 4: Distancia
$$d = \frac{|3(-8) + 4(-1) - 5(-5) - 6|}{\sqrt{3^2 + 4^2 + (-5)^2}}$$ $$= \frac{|-24 - 4 + 25 - 6|}{\sqrt{9 + 16 + 25}} = \frac{|-9|}{\sqrt{50}}$$ $$= \frac{9}{5\sqrt{2}} = \frac{9\sqrt{2}}{10} \approx 1.27$$
✅ Distancia recta-plano: $d = \frac{9\sqrt{2}}{10} \approx 1.27$ unidades

Ejercicio 21 - Verdadero o Falso

Responda falso o verdadero a las siguientes preguntas, justifique su respuesta.

📝 Desarrollo:

a) Dos rectas diferentes se pueden cortar exactamente en dos puntos.
FALSO
Justificación: En geometría euclidiana, dos rectas diferentes solo pueden:
• No intersectarse (paralelas o alabeadas)
• Intersectarse en exactamente UN punto
• Ser coincidentes (infinitos puntos)
Nunca pueden cortarse en exactamente dos puntos.
b) Si el producto vectorial entre dos vectores es igual al vector nulo, uno de los dos vectores tiene que ser el vector nulo.
FALSO
Justificación: Si $\vec{u} \times \vec{v} = \vec{0}$, significa que los vectores son paralelos (o uno es el vector nulo). Por ejemplo: $\vec{u} = (2, 4, 6)$ y $\vec{v} = (1, 2, 3)$ son paralelos pero ninguno es nulo, y su producto cruz es cero.
c) El producto escalar de un vector por el mismo, es su magnitud elevada al cuadrado.
VERDADERO
Justificación: $\vec{u} \cdot \vec{u} = \|\vec{u}\|^2$ por definición del producto escalar.
d) Una recta en el espacio se determina mínimo con 3 puntos.
FALSO
Justificación: Una recta se determina con DOS puntos distintos o con un punto y un vector director.
e) El producto escalar entre dos vectores siempre da como resultado un número entero.
FALSO
Justificación: El producto escalar puede ser cualquier número real. Por ejemplo: $\vec{u} = (1.5, 2.3)$ y $\vec{v} = (1, 1)$ da $\vec{u} \cdot \vec{v} = 3.8$.
f) Dos planos diferentes se pueden cortar en infinitos puntos.
VERDADERO
Justificación: Cuando dos planos se intersectan (no son paralelos ni coincidentes), su intersección es una recta, que contiene infinitos puntos.
✅ Respuestas:
a) FALSO   b) FALSO   c) VERDADERO
d) FALSO   e) FALSO   f) VERDADERO
